# **FUDU – Multi-Vendor Food Delivery Platform**

## **Overview**

FUDU is a multi-vendor food delivery web application inspired by platforms like Swiggy. It connects customers with multiple restaurants and enables seamless ordering, delivery tracking, and secure payments.

## **Features**

* **Multi-Vendor Support** – Browse and order from various restaurants
* **User Authentication** – Secure login and registration (JWT-based)
* **Restaurant Browsing** – Search, filter, and view menus
* **Shopping Cart** – Add/remove items and manage selections
* **Order Tracking** – Real-time order status updates
* **Secure Payments** – Integrated payment gateway
* **User Profiles** – Manage addresses & order history
* **Reviews & Ratings** – Rate restaurants and dishes

## **Tech Stack**

* **Frontend:** React.js, HTML5, CSS3
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **API Architecture:** RESTful API

## **Installation**

```bash
# Clone the repository
git clone <repository-url>
cd fudu

# Frontend setup
cd frontend
npm install
npm start

# Backend setup
cd ../backend
npm install
npm run dev
```
